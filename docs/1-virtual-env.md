# virtual env

- ref: <https://docs.python.org/ja/3.11/tutorial/venv.html>

## init venv

```bash
python3 -m venv .
```

## activate venv

```bash
source bin/activate
```

## install packages

```bash
python -m pip install novas
```

## freeze packages

```bash
python -m pip freeze > requirements.txt
```

## install dependencies

```bash
pip install -r requirements.txt
```
